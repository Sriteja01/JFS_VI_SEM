git commands


first time:

1. git init
2. git add . 
3. git commit -m "message"
4. git remote add origin remote-url
5. git push origin main
6. you can check in remote repository

second time
if you can add any files in local repository or you can change the existing files in local repo

1. git status
2. git add . 
3. git commit -m "message"
4. git push origin main

1. rmdir /s /q .git
